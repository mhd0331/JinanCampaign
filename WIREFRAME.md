## 주요 인포그래픽 요소

### 1. 정책카드 내 미니차트
각 6대 공약 카드에는 해당 정책의 핵심 데이터를 보여주는 작은 차트가 포함:
- 주민참여행정: 찐반장 프로그램 대상 연령대 (파이차트)
- 복지: 장애인 복지 개선 분야별 임팩트 (막대차트) 
- 경제: 지역화폐 발행 계획 (카드 형태)
- 행정: DRT 서비스 커버리지 (막대차트)
- 인프라: 인터넷 속도 개선 전후 비교 (막대차트)
- 인구: 연령대별 지원금 (막대차트)

### 2. 현황 데이터 섹션
- **통계 카드**: 주요 지표 4개 (인구, 행정구역, 고령화율, 총예산)
- **인구변화 추이**: 2020-2024년 라인차트
- **연령대별 분포**: 파이차트 (0-14세, 15-64세, 65세+)
- **예산 배분**: 6대 공약별 컬러코딩된 막대차트
- **면별 현황**: 사업 개수와 예산의 복합 막대차트

### 3. 경제효과 시뮬레이션
- **지역화폐 순환**: 시간에 따른 유통액, 참여업체, 일자리 증가 (에어리어차트)
- **스마트팜 효과**: 참여자별 소득 증대 효과 (막대차트)
- **신재생에너지**: 발전 유형별 수익 분배 (리스트 형태)
- **종합 효과**: 핵심 지표 4개 대형 표시

## 데이터 소스

모든 차트와 인포그래픽은 제공된 캠페인 문서의 실제 데이터를 기반으로 제작:

1. **6대통합공약최종.md**: 공약별 세부 내용과 예산
2. **면별공약요약최종.md**: 11개 면별 프로젝트와 예산
3. **선거공약-0611.md**: 구체적인 정책 실행 방안
4. **후보자 이력**: 프로필 정보와 경력

## 기술적 구현

- **차트 라이브러리**: Recharts (React 기반)
- **스타일링**: Tailwind CSS + 커스텀 진안 테마 컬러
- **반응형 디자인**: 모바일/태블릿/데스크톱 최적화
- **AI 챗봇**: OpenAI API 연동으로 공약 관련 Q&A
- **데이터 관리**: TypeScript 기반 타입 안전성 보장

## 사용자 경험 플로우

1. **입장**: 히어로 섹션에서 임팩트 있는 첫인상
2. **탐색**: 내비게이션으로 관심 섹션 바로 이동
3. **이해**: 시각적 데이터로 공약 내용 직관적 파악
4. **상담**: AI 챗봇으로 궁금한 점 즉시 해결
5. **참여**: 문의 폼으로 직접 소통 채널 제공

이 구조를 통해 유권자들이 후보의 공약을 데이터 기반으로 명확히 이해하고, 각 지역별 맞춤 정책을 쉽게 확인할 수 있도록 설계되었습니다.
